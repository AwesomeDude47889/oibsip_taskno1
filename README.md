# Unit Converter App

## Objective

The objective of this project is to develop a Unit Converter Android application that allows users to convert values between different measurement units quickly and accurately.

The application supports conversions in the following categories:

* Weight
* Distance

This project was developed as part of the Oasis Infobyte Android App Development Internship.

---

## Features

### Weight Conversion

* Grams
* Kilograms
* Pounds
* Ounces

### Distance Conversion

* Millimeters
* Centimeters
* Meters
* Kilometers
* Inches
* Feet
* Miles

### Additional Features

* User-friendly interface
* Real-time conversion results
* Input validation
* Dynamic unit selection using Spinners

---

## Tools & Technologies Used

* Android Studio
* Java
* XML
* Android SDK

---

## Project Structure

* XML Layout for User Interface
* Java for Conversion Logic
* Spinner Components for Unit Selection
* Button for Conversion Execution
* TextView for Result Display

---

## Working

1. Enter a value.
2. Select a category (Weight or Distance).
3. Choose the unit to convert from.
4. Choose the unit to convert to.
5. Click the Convert button.
6. The converted result is displayed instantly.

---

## Outcome

The application successfully converts values between various weight and distance units while providing a simple and user-friendly Android experience.
